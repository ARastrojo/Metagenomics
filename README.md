#### Install VirtualBox Guest Additions


sudo apt-get update  
sudo apt-get install gcc  
sudo apt-get install make  

Go to menu bar, click on Devices -> Insert Guest additions CD  
In temrinal:  
```
cd /media/${USER}/VBox_GAs_7.0.6  
sudo ./VBoxLinuxAdditions.run  
```
Reboot system.
